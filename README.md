# heart-disease-classification
Heart disease classification using machine learning
This project aims to classify whether a patient has heart disease or not using machine learning models such as Logistic Regression, KNN, and Decision Tree.

Workflow / Steps :
1-Data preprocessing
2-Train/test split
3-Model training
4-Model comparison
5-Hyperparameter tuning
6-Confusion matrix evaluation

Models Used :
Logistic Regression
KNN
Decision Tree

Final Result :
Logistic Regression achieved the best overall performance among the evaluated classifiers.
